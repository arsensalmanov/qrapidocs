title: "Authentication"
description: "How to use Bearer authentication for API requests"
---

<img src="/images/authentication.png" alt="Hero Dark" />

To authenticate, you need to obtain an API token. You can get your API token by visiting your [dashboard](https://qrapi.app/dashboard). The token serves as a credential to authorize API requests.

<Steps>
  <Step title="Obtain the API token">
    Obtain the API token from your [dashboard](https://qrapi.app/dashboard).
  </Step>
  <Step title="Use it in your API requests">
    Include the API token in the `Authorization` header of your API requests
    using the Bearer scheme - `Authorization: Bearer API_TOKEN`.
  </Step>
</Steps>

### Example

```
curl "https://api.qrapi.app/v1/qr/create?text=Hello%2C+World%21" \
  -H 'Content-Type: application/json' \
  -H "Authorization: Bearer <API_TOKEN>"
```
